# TeamFlow

## Project Workspace & Workforce Management Platform

TeamFlow is a mobile-first Progressive Web Application (PWA) designed to help organizations manage interns, employees, projects, tasks, attendance, and team collaboration through a centralized platform.

The platform is built around Project Workspaces, allowing organizations to manage multiple projects simultaneously while providing role-based access for administrators and team members.

## Project Overview

Organizations often manage multiple teams, projects, interns, and employees simultaneously. Traditional approaches using spreadsheets, emails, and messaging applications can lead to poor visibility, communication gaps, and inefficient task tracking.

TeamFlow addresses these challenges by providing a centralized workspace where administrators can manage projects, assign tasks, monitor attendance, track progress, and communicate important updates while employees and interns can collaborate and monitor their work effectively.

## Installation

Clone the repository:

```bash
git clone https://github.com/AaliyaPatel-11/teamflow.git
```

Navigate to the project directory:

```bash
cd teamflow
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
